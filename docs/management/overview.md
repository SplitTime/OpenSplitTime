# Managing an Event: Overview

Now that your event group is set up and staged, you are ready to prepare for the event and learn how to manage it.

This guide will cover the following steps:

1. Preparing for your event
2. Checking in and starting entrants
3. Monitoring entrants
4. Entering and editing times
5. Winding up your event
