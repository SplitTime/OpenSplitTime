# Winding Up

## Setting Stops

At the conclusion of your Event, you may have Entrants still showing as "in progress" even though you
know these Entrants are no longer on the course. This occurs if Entrants drop and leave the Course without
notifying timing crews. It may also occur if timing crews are not accurately reporting stopped Entrants.

To set stops for all Entrants in an Event at the last point for which times were recorded, use the Establish
Drops tool.

1. Visit the Construction page (**Admin > Construction**)

1. If the Event Group has multiple Events, locate the applicable Event

1. Select **Actions > Establish Drops** from the menu for the applicable Event

1. Click **OK** to confirm your choice

OpenSplitTime will set a stop at the last recorded Split Time for each Entrant.

## Disabling Live Entry

When you are finished using OST Remote and editing times using the Live Entry screen, be sure to disable live
entry for the Event Group. Disabling live entry will ensure the Event Group does not appear as an option when an
Owner or Steward logs in to OST Remote. This can avoid the problem of an OST Remote operator inadvertently
selecting the wrong Event Group when attempting to enter times for a future event.

Disable live entry from the Construction screen (**Admin > Construction > Group Actions > Disable Live**).

## Exporting Data

OpenSplitTime provides tools for exporting your results in different formats, including:

1. Full results

1. Finisher list

1. ITRA results format

1. Ultrasignup results format

## Exporting Full Results

You can export full results to a CSV file from the Full Results screen (**Results > Full > Export
Spreadsheet**). Time data will be exported in the same format it is displayed on screen (Elapsed, AM/PM,
24-Hour, or Segment).

**This export option is available only after the Event is concluded** (that is, after all Entrants
who started have either finished or dropped).

## Exporting a Finisher List

You can export a summary finisher list to a CSV file. Access the tool from the Construction page
(**Admin > Construction > Actions > Export Finisher List**).

This export option is available at any time during or after the Event.

## Exporting ITRA Results

You can export results to ITRA format. Access the tool from the Construction page
(**Admin > Construction > Actions > Export ITRA**).

This export option is available at any time during or after the Event, but it includes only finished runners.

## Exporting Ultrasignup Results

You can export results to a CSV file in a format that is readable by Ultrasignup. Access the tool from the
Construction page (**Admin > Construction > Actions > Export to Ultrasignup**).

**This export option is available only after the Event is concluded** (that is, after all Entrants
who started have either finished or dropped).
