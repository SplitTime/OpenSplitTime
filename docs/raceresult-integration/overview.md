# RaceResult Integration Overview

This guide will walk you through the steps to connect a RaceResult event to OpenSplitTime using RaceResult's webhook functionality. By following the instructions below, you can set up a live data feed from RaceResult to OpenSplitTime, allowing you to leverage OpenSplitTime's features while using RaceResult for timing and data management.

## Steps Overview

This guide will cover the following steps:

1. Opening the target event in RaceResult
2. Setting the event name to match OpenSplitTime
3. Configuring timing points to match OpenSplitTime aid stations
4. Connecting and mapping decoders to timing points
5. Creating an exporter to send data to OpenSplitTime
6. Activating the exporter to start sending live data
7. Confirming that data is being delivered to OpenSplitTime

For full details, see the [Webhook Configuration](webhook-setup.md) guide.
