# RunSignup Integration

[RunSignup](https://runsignup.com){:target="_blank"} is a race registration platform that can also provide participant and results data to OpenSplitTime.

This section explains how to connect OpenSplitTime to RunSignup and how to obtain the API credentials needed for that connection.
